4️⃣ Backend API Route (/api/scan)

Create POST API route that:

Accepts:
{
extractedText: string,
allergies: string[],
language: string
}

Implements allergen detection logic:

Logic:

If extracted text contains any selected allergen → NOT SAFE

If extracted text contains "may contain" or "processed in facility" → CAUTION

Else → SAFE

Return:

{
status: "SAFE" | "CAUTION" | "NOT SAFE",
matchedAllergen: string | null,
message: localizedMessage
}
